# ❌⭕ Tic Tac Toe Game

A simple and interactive **Tic Tac Toe Web Application** built using **HTML, CSS, and JavaScript**.

This game allows two players to play turn-based Tic Tac Toe in a clean and responsive interface.

---

## 📌 Project Overview

The Tic Tac Toe Game is a classic 2-player game where users take turns marking spaces in a 3×3 grid. The player who succeeds in placing three of their marks in a row (horizontal, vertical, or diagonal) wins the game.

This project demonstrates:
- DOM manipulation  
- Game logic implementation  
- Event handling  
- Frontend UI design  

---

## ✨ Features

- ❌⭕ Two-player gameplay  
- 🔄 Turn-based system  
- 🏆 Winner detection  
- 🔁 Restart game option  
- ⚡ Instant response on clicks  
- 📱 Simple and responsive UI
